2016流水账

发表于 2017-01-28   |   分类于 生活   |  

转眼间一年的时间又过了，看着外面的烟火，虽然沉浸在喜悦的气氛当中，但是心中不由生出一丝惶然。这一年到底干了些什么？新的一年会不会更好？于是我想记录下去年的经济状态，以便日后对比。


去年年头我换了一份工作，工资也骤然飙升。似乎也验证了IT行业，最快的涨薪方式就是跳槽这句话。加上空余时间帮朋友做了些私活，工资实际每月到手也差不多10K了。这些工资虽不算多，但是对于武汉这个二线城市来讲已经算是满足了。大概是3月份入的职，到年末有一个月的年终奖，算上去是拿了11月的工资。
工作以来，我对于日常的花费基本很少克制，虽然生活总本可以省下许多钱，但是我一直理解的是，钱绝对不是省下来的，没有赚钱的能力，再怎么去节约也无济于事。牺牲生活的品质去攒钱是不可取的，当然一切的消费必须理性，认清自己需要什么更为重要。
每个月850的房租加上1000左右的伙食费，基本一年的消费2W多就消耗在这儿了。比较高兴的是给家里消费了不少，其中包括自驾游的消费，给爸妈各买了一个手机，给家里买了一个二手IPAD，一个麦克风，过年给家里买2000的衣服，给奶奶2000，再加上平时的一些琐碎，实际消费应该是超出了1W。自己平时酷爱数码产品，算上手机大概消费接近8K左右。在上班的地方基本没有什么娱乐消费，基本去的最多就是网吧打游戏了。每次上网的消费大概在50左右，一周最少一次，大约估摸了下一年的网费2K。除了这些消费，再算比较大头的就是和朋友们聚会，每次消费至少300以上，不管是家里的同学还是武汉这边的。因为自己是一个很难对人说NO的人，所以导致这部分消费有些不合理，以后会尽量克制。算上打牌输了2K，朋友之间的消费应该在5K+。还有一些买衣服，出去玩之类的其他消费。

名称 金额 描述
房租 10K 房租加上水电费
伙食 10K 日常吃饭
家庭 10K 给家里买东西
上网 2K 平时上网打游戏
数码 8K 手机，VR，电脑配件…
朋友 5K 包括吃饭，打牌…
其他 15K 自己也不知道怎么用的，包括买衣服，零食，出去玩…
存余 50K 一年到头所有的存款，一分不剩。

看了下支付宝交易，其中大部分网购用于了数码产品
